# 每日自动化任务清单

## 每天早上 9 点，和我说：

**"生成今天的小红书文案"**

我会自动：
1. 生成当天的文案（含 3 个标题备选）
2. 提供配图建议
3. 告诉最佳发布时间
4. 提供私信引流话术

---

## 你只需要做：

### 上午（10-12 点）
- [ ] 选 1 个标题
- [ ] 制作配图（Canva/稿定设计，10 分钟）
- [ ] 检查文案（修改个人化内容）

### 下午（18-20 点）
- [ ] 发布笔记
- [ ] 回复评论（前 1 小时很重要）
- [ ] 私信引流（有咨询就导微信）

### 晚上（20-21 点）
- [ ] 朋友圈发 3 条（我帮你准备）
- [ ] 跟进微信咨询

---

## 每周日晚上

和我说："生成下周的文案"
我会一次性生成 7 篇，你可以批量制作配图

---

## 需要调整时

告诉我：
- "换个风格"（更正式/更活泼）
- "换个主题"（小红书卖货/闲鱼卖货）
- "加个钩子"（强化引流）
- "生成朋友圈文案"（配套内容）
