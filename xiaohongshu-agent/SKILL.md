# 小红书文案自动生成 Skill

## 触发条件

用户请求生成小红书内容时：
- "生成今天的小红书文案"
- "写篇笔记"
- "准备内容"
- "今天发什么"
- "帮我写文案"

## 业务背景

**定位：** 副业教学（AI 代写、小红书卖货、闲鱼卖货）

**目标人群：** 想搞副业的上班族、宝妈、学生党

**人设：** 副业实战教练，有成熟交付团队

**引流目标：** 私信→微信→成交（¥699/¥1280/¥2980）

## 文案类型轮换

每天生成不同类型，避免重复：

| 类型 | 比例 | 目的 |
|------|------|------|
| 学员案例 | 30% | 建立信任、展示结果 |
| 干货教程 | 30% | 展示专业、收藏率高 |
| 避坑指南 | 20% | 引发共鸣、互动高 |
| 工具资源 | 20% | 引流效果好 |

## 爆款文案结构

### 标题公式

1. **数字 + 结果**："7 天，从 0 到出第一单"
2. **身份 + 反差**："上班族，副业月入 5000+"
3. **痛点 + 方案**："被限流？可能是这 3 个原因"
4. **稀缺 + 行动**整理了 30 个平台，新手也能用"

### 正文结构

```
【吸引注意的开头】
- 结果展示/痛点引入/问题提出

【核心内容】
- 分点列出（3-5 点最佳）
- 每点有具体说明
- 用 emoji 增加可读性

【行动呼吁】
- 评论区互动（扣"666"、"求资料"）
- 引导看主页/私信
- 点赞收藏提醒

【标签】
- 3-5 个大标签（#副业 #赚钱）
- 2-3 个垂直标签（#AI 代写 #写作变现）
```

## 输出格式

每次生成提供：

1. **3 个标题备选**（让用户选）
2. **完整正文**（含 emoji）
3. **标签建议**
4. **发布时间建议**（18-21 点或 12-13 点）
5. **AI 配图提示词**（Midjourney/SD 可用，含封面 + 内页 3-5 张）

## 优化原则

参考小红书爆款特点：

1. **封面重要**：标题文字醒目、对比度高
2. **前 3 行关键**：决定用户是否继续阅读
3. **emoji 适度**：每段 1-2 个，增加可读性
4. **段落简短**：每段 2-3 行，避免大段文字
5. **互动引导**：结尾必须有行动呼吁

## 注意事项

- 别直接留微信（会被限流）
- 用"看主页置顶"、"私信"引导
- 避免敏感词（赚钱→变现、收入→收获）
- 保持真实感（别夸大收益）
- 每天内容类型轮换
