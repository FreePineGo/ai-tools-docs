## 📝 工作流程

### 每轮执行 (10 分钟)

1. **功能测试** (5 分钟)
   - 运行 `python main.py --stock 600519`
   - 检查报告生成
   - 捕获异常和错误

2. **代码审查** (3 分钟)
   - 查看 git 变更
   - 检查新文件
   - 审查代码修改

3. **用户体验检查** (2 分钟)
   - 评估命令行输出
   - 检查错误提示
   - 测试响应速度

4. **生成任务计划**
   - 更新 `task_plan.md`
   - 记录测试结果
   - 列出新问题

5. **分配任务**
   - 更新三位员工的任务清单
   - 设置优先级和截止时间

## 🔧 管理命令

### 查看 Cron 状态
```bash
openclaw cron list
```

### 手动触发任务
```bash
openclaw cron run <job-id>
```

### 暂停任务
```bash
openclaw cron update <job-id> --enabled=false
```

### 恢复任务
```bash
openclaw cron update <job-id> --enabled=true
```

### 删除任务
```bash
openclaw cron remove <job-id>
```
