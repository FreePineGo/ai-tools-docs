# AI 工具对比表

## 主流 AI 工具功能对比

| 工具名称 | 核心功能 | 优势 | 劣势 | 适用场景 |
|---------|---------|------|------|---------|
| **ChatGPT** | 对话、写作、代码、分析 | - 通用能力强<br>- 插件生态丰富<br>- 多模态支持 | - 付费墙限制<br>- 知识截止限制 | 日常助手、内容创作、编程辅助 |
| **Claude** | 长文本处理、写作、分析 | - 上下文窗口大<br>- 写作质量高<br>- 安全性好 | - 代码能力稍弱<br>- 多模态有限 | 文档分析、长文写作、研究辅助 |
| **Gemini** | 多模态、搜索集成、代码 | - Google 生态集成<br>- 免费额度高<br>- 实时搜索 | - 输出质量波动<br>- 隐私顾虑 | 研究查询、多模态任务、Google 用户 |
| **Perplexity** | 搜索增强、研究、引用 | - 实时网络搜索<br>- 自动引用来源<br>- 专业模式 | - 创作能力有限<br>- 依赖网络质量 | 学术研究、事实核查、新闻追踪 |
| **Cursor** | 代码编辑、项目理解 | - IDE 集成<br>- 代码库理解<br>- 自动补全 | - 仅限编程<br>- 学习曲线 | 软件开发、代码审查、重构 |
| **Midjourney** | 图像生成、艺术设计 | - 图像质量顶尖<br>- 艺术风格丰富<br>- 社区活跃 | - 需 Discord<br>- 付费使用<br>- 文字渲染弱 | 视觉设计、概念艺术、营销素材 |
| **DALL-E 3** | 图像生成、集成 ChatGPT | - 文字渲染好<br>- ChatGPT 集成<br>- 易用性高 | - 风格较单一<br>- 控制精度低 | 快速原型、社交媒体、演示文稿 |
| **Stable Diffusion** | 图像生成、本地部署 | - 开源免费<br>- 高度可定制<br>- 本地运行 | - 配置复杂<br>- 需要硬件<br>- 质量不稳定 | 批量生成、隐私敏感、定制需求 |
| **Notion AI** | 笔记增强、文档管理 | - Notion 集成<br>- 模板丰富<br>- 团队协作 | - 功能有限<br>- 依赖 Notion | 知识管理、会议记录、项目文档 |
| **Jasper** | 营销文案、SEO 内容 | - 营销模板多<br>- SEO 优化<br>- 品牌声音 | - 价格较高<br>- 通用性弱 | 营销团队、广告文案、博客写作 |
