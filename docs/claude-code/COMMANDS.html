命令行参数

基本命令

| 命令 | 说明 | 示例 |

|------|------|------|

| claude | 启动交互式会话 | claude |

| claude --version | 显示版本信息 | claude --version |

| claude --help | 显示帮助信息 | claude --help |

| claude [prompt] | 执行单条指令 | claude "解释这个文件" |

配置选项

| 参数 | 简写 | 说明 |

|------|------|------|

| --model | -m | 指定使用的模型 |

| --max-tokens | -t | 设置最大 token 数 |

| --timeout | -T | 设置超时时间（毫秒） |

| --config | -c | 指定配置文件路径 |

| --verbose | -v | 详细输出模式 |

| --quiet | -q | 静默模式 |

| --json | -j | JSON 格式输出 |

| 参数 | 说明 |

文件引用语法

单文件引用

@filename           # 当前目录文件

@path/to/file       # 指定路径文件


@./relative/path    # 显式相对路径


@/absolute/path     # 绝对路径

多文件引用

@*.ts               # 所有 TypeScript 文件

@src/**/*.js        # src 下所有 JS 文件


@**/*.test.ts       # 所有测试文件


@{file1,file2}      # 多个指定文件

文件夹引用

@src/               # src 目录

@components/**      # components 及其子目录


@docs/*             # docs 直接子项（不递归）
