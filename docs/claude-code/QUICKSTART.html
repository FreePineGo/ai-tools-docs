Claude Code 快速开始

5 分钟快速上手

第一步：启动 Claude Code

进入项目目录


cd /path/to/your/project



启动交互式会话


claude

第二步：基本对话

启动后，你可以直接输入自然语言指令：

你好，请帮我创建一个简单的 Python Hello World 程序

第三步：查看和执行建议

Claude Code 会：

1. 分析你的请求

2. 生成代码或命令

3. 等待你的确认

4. 执行操作
